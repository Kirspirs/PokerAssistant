# 🃏 Poker Assistant (Pre-flop Edition)



> **Аналитический инструмент для игры в покер.** Помогает принимать математически обоснованные решения на этапе пре-флопа и структурировать данные флопа.

---

## 🚀 Текущий функционал (MVP)

Программа уже умеет обрабатывать первый раунд торгов (**Pre-flop**) и подготавливать данные для флопа:

*   **🔢 Ранжирование номиналов**: Встроена система весов (Dictionary) от `2` до `A` (14), что позволяет проводить сравнение любой сложности.
*   **📂 Интеллектуальный парсинг**: Скрипт корректно разделяет пользовательский ввод (например, `Ah Ks`) на **номинал** (`A`, `K`) и **масть** (`h`, `s`).
*   **📊 Стратегическая группировка**:
    *   **Tier-1 & Tier-2**: Автоматическое определение топовых рук для агрессивной игры (Рейз).
    *   **Suited Logic**: Учет одномастности для принятия решения по средним картам (Группа 3).
    *   **Pocket Pairs**: Специальная логика для оценки мелких карманных пар (от 22 до 77).
*   **📥 Сбор данных Флопа**: Реализован механизм захвата первых трех общих карт стола для последующего анализа комбинаций.
